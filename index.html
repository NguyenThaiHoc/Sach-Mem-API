Đặt tên biến

  • Trong html
    - Với id, sử dụng dấu gạch chân để nối giữa hai từ, tên đặt trong dấu nháy kép
    Ví dụ: id="example_one"

    - Với class, sử dụng dấu trừ để nối giữa hai từ, tên đặt trong dấu nháy kép
    Ví dụ: class="example-one"
  • Trong javaScript
    Viết hoa chữ cái đầu tiên của các từ sau từ thứ nhất.
    Ví dụ: exampleOne
    Viết hoa toàn bộ với biến toàn cục.
    Ví dụ: EXAMPLEONE
  • Trong css

Angular

Các thuộc tính của của Angular ng-.. phải đặt lên trước sau đó đến id rồi class và các thuộc tính data-..
Lí do: Sẽ tiết kiệm thời gian sau này đọc lại code
Ví dụ với 1 div có ng-if chỉ cần đọc xong ng-if là có thể biết được có phải đọc phần còn lại của div hay không.

Lùi đầu dòng space 2 trong code

Mỗi dòng không quá 80 kí tự, với code html nếu một dòng có nhiều thuộc tính, làm dòng dài hơn 80 kí tự thì xuống dòng với mỗi thuộc tính, thuộc tính tiếp theo để thẳng với thuộc tính thứ nhất.

Kết thúc mỗi function, mỗi module, mỗi khối lệnh phải có xuống dòng, tức là cách 1 dòng với khối lệnh tiếp theo.

Kết thúc mỗi file code phải xuống 1 dòng, tức là để trống 1 dòng cuối cùng.

interactRadio.letCheck(); -> Kiểm tra kết quả, return true/false
